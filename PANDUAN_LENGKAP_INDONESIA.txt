================================================================================
            BAGIAN 3: PERUBAHAN DARI PROJECT LAMA KE BARU
================================================================================

1. DATABASE: PostgreSQL â†’ MongoDB
   --------------------------------
   LAMA (PostgreSQL):
   - Table-based dengan foreign keys
   - Pakai Prisma ORM
   - Hosted di Supabase
   
   BARU (MongoDB):
   - Document-based (JSON-like)
   - Pakai Mongoose ODM
   - Bisa local atau MongoDB Atlas
   - Data nested/embedded lebih fleksibel

2. BACKEND: Next.js API Routes â†’ Express.js
   ------------------------------------------
   LAMA (Next.js):
   - File: /app/api/register/route.ts
   - Server-side rendering
   - Tightly coupled dengan frontend
   
   BARU (Express.js):
   - File: /server/src/controllers/authController.js
   - Pure REST API
   - Separated dari frontend
   - Bisa dipakai untuk mobile app juga

3. FRONTEND: Next.js â†’ React.js
   -----------------------------
   LAMA (Next.js):
   - Server components
   - File-based routing (/app/page.tsx)
   - Built-in API routes
   
   BARU (React.js):
   - Client-side rendering
   - React Router (/src/App.jsx)
   - API calls via Axios ke backend terpisah

4. AUTHENTICATION: NextAuth â†’ JWT
   --------------------------------
   LAMA (NextAuth):
   - Session-based
   - Pakai cookies
   
   BARU (JWT):
   - Token-based
   - Stored di localStorage
   - Header: Authorization: Bearer <token>

CONTOH KONVERSI DATA FETCHING:

1. GET ALL EMPLOYEES
   ------------------
   LAMA (Next.js + Supabase):
   const { data } = await supabase
     .from('employees')
     .select('*');
   
   BARU (React + Axios):
   const response = await api.get('/employees');
   const employees = response.data.data;

2. CREATE JOB
   ----------
   LAMA (Next.js + Prisma):
   const job = await prisma.job.create({
     data: { title, type, ... }
   });
   
   BARU (React + Axios):
   const response = await api.post('/jobs', {
     title, type, ...
   });
   const job = response.data.data;

3. GET JOB WITH ASSIGNMENTS
   -------------------------
   LAMA (Next.js + Prisma):
   const job = await prisma.job.findUnique({
     where: { id },
     include: { job_assignments: { include: { employees: true } } }
   });
   
   BARU (React + Axios):
   const response = await api.get(`/jobs/${id}`);
   const job = response.data.data; // Sudah include assignments
