LANGKAH 1: INSTALL & JALANKAN BACKEND
--------------------------------------
1. Buka terminal/PowerShell, masuk ke folder server:
   cd server

2. Install dependencies (jika belum):
   npm install

3. Setup environment variables (jika belum):
   copy .env.example .env
   
   Edit file .env jika perlu (default sudah OK untuk MongoDB lokal)

4. Pastikan MongoDB berjalan:
   - Lokal: Service MongoDB harus running
   - Atlas: Copy connection string ke MONGODB_URI di .env

5. Jalankan backend:
   npm run dev

   ‚úÖ Backend running di http://localhost:5000
   ‚úÖ Jika muncul "MongoDB Connected" berarti berhasil!


LANGKAH 2: INSTALL & JALANKAN FRONTEND
---------------------------------------
1. Buka terminal/PowerShell BARU, masuk ke folder client:
   cd client

2. Install dependencies (jika belum):
   npm install

3. Jalankan frontend:
   npm run dev

   ‚úÖ Frontend running di http://localhost:5173
   ‚úÖ Atau jika port 5173 sudah dipakai, akan otomatis pakai port lain (misal 3001)

4. Buka browser:
   http://localhost:5173  (atau port yang ditampilkan di terminal)

================================================================================
                    TROUBLESHOOTING
================================================================================

‚ùå Error: ECONNREFUSED ::1:27017
   SOLUSI: MongoDB belum berjalan
   - Windows: Cek service MongoDB di Task Manager
   - Atau: Pastikan connection string di .env benar

‚ùå Error: CORS policy blocked
   SOLUSI: Port frontend tidak sesuai dengan backend
   - Cek CLIENT_URL di server/.env
   - Default: http://localhost:5173
   - Jika frontend di port lain (misal 3001), ubah CLIENT_URL

‚ùå Error: Port already in use
   SOLUSI: Port sudah dipakai aplikasi lain
   - Backend: Ubah PORT di server/.env
   - Frontend: Vite akan otomatis pakai port lain

‚ùå Login tidak redirect ke dashboard
   SOLUSI: Refresh browser (Ctrl+R atau F5)
   - Code frontend mungkin belum reload

‚ùå Data tidak muncul / API error
   SOLUSI: Cek backend logs di terminal
   - Lihat error message
   - Pastikan MongoDB connected
